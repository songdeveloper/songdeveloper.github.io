iOSејҖеҸ‘дёӯзҡ„йҹіи§Ҷйў‘

еҸ‘иЎЁдәҺ 2017-01-04
ејҖеҸ‘йҹід№җж’ӯж”ҫеҷЁж—¶йҒҮеҲ°зҡ„й—®йўҳпјҡ

1пјҢеҪ“еңЁдҪҝз”Ёе…¶д»–йҹід№җж’ӯж”ҫиҪҜд»¶ж’ӯж”ҫйҹід№җж—¶пјҢиҝӣе…ҘAPPпјҢжңӘжү“ејҖйҹід№җж’ӯж”ҫжЁЎеқ—пјҢжӯӨж—¶еӨ–йқўз¬¬дёүж–№йҹід№җж’ӯж”ҫеҷЁеҒңжӯўж’ӯж”ҫгҖӮ
йңҖжұӮпјҡдҪҝз”Ёз¬¬дёүж–№ж’ӯж”ҫйҹід№җиҪҜд»¶ж—¶пјҢжӯӨж—¶иҝӣе…ҘAPPжөҸи§Ҳе…¶д»–еҶ…е®№пјҢеҰӮжһңдёҚжү“ејҖйҹід№җж’ӯж”ҫжЁЎеқ—пјҢеҲҷдёҚеҒңжӯўз¬¬дёүж–№йҹід№җиҪҜд»¶зҡ„дҪҝз”ЁгҖӮ
еҗҢзұ»дә§е“ҒжөӢиҜ•пјҡе…¶д»–еҺӮе•Ҷйҹід№җж’ӯж”ҫиҪҜд»¶пјҢеҸ‘зҺ°д»ҺзҪ‘жҳ“дә‘йҹід№җиҝӣе…ҘеҲ°QQйҹід№җпјҢдјҡжҡӮеҒңпјҢдҪҶжҳҜдҪҝз”Ёж–—йұјж—¶еҸ‘зҺ°пјҢд»Һе…¶д»–йҹід№җж’ӯж”ҫиҪҜд»¶иҝӣе…ҘпјҢжңӘе…ій—ӯгҖӮ
й—®йўҳеҺҹеӣ пјҡжҹҘзңӢд»Јз Ғж–ҮжЎЈпјҢеҸ‘зҺ°е…ій”®еңЁдәҺиҝҷдёҖиЎҢд»Јз Ғ[[AVAudioSession sharedInstance] setActive:YES error:nil]; е®ғжҺ§еҲ¶зқҖйҹід№җж’ӯж”ҫеҷЁзҡ„жҝҖжҙ»пјҢжү§иЎҢж—¶жңәиҝҮж—©пјҢеҜјиҮҙе…¶е®ғз¬¬дёүж–№йҹід№җиҪҜд»¶иў«е…ій—ӯ
и§ЈеҶіеҠһжі•пјҡи§ҰеҸ‘APPеҶ…йҹід№җжЁЎеқ—ж’ӯж”ҫеҠҹиғҪпјҢжүҚи°ғз”ЁдёҠйқўжҝҖжҙ»д»Јз Ғ

2пјҢж’ӯж”ҫзҪ‘з»ңйҹід№җпјҢеҸ‘зҺ°зҪ®дәҺеҗҺеҸ°зҡ„жғ…еҶөдёӢпјҢдёҖдјҡйҹід№җиҮӘе·ұе…ій—ӯпјҢйҮҮз”ЁеҰӮдёӢеҠһжі•пјҡз”іиҜ·дәҶдёҖдёӘеҗҺеҸ°д»»еҠЎID
// дёӢйқўзҡ„д»Јз Ғдё»иҰҒжҳҜдёәдәҶйҳІжӯўAPPиҝӣе…ҘеҗҺеҸ°еҗҺпјҢз”іиҜ·дёҖдёӘеҗҺеҸ°IDпјҢйҳІжӯўеҗҺеҸ°ж’ӯж”ҫйҹід№җиў«жҡӮеҒң

3пјҢдҪҝз”ЁAVPlayerж’ӯж”ҫдёҖдёӘиҝңзЁӢmp3ең°еқҖпјҢзӣ‘еҗ¬playItemзҡ„statusеҸ‘зҺ°еҪ“е·Із»ҸеҮҶеӨҮеҘҪж’ӯж”ҫж—¶пјҢи°ғз”ЁAVPlayerзҡ„playж–№жі•ж’ӯж”ҫпјҢйңҖиҰҒзӯүеҫ…еҘҪй•ҝж—¶й—ҙжүҚиғҪж’ӯж”ҫгҖӮи§ЈеҶіеҠһжі•пјҡ

е»әи®®ж–Үз« пјҡ
  1. гҖҠMP3зј“еҶІ2MBжүҚејҖе§Ӣж’ӯж”ҫзҡ„и§ЈеҶіж–№жі•гҖӢ
  2. гҖҠе®һзҺ°иҫ“еҮәh264зӣҙж’ӯжөҒзҡ„rtmpжңҚеҠЎеҷЁгҖӢ
  3. гҖҠiOSйҹійў‘з ”з©¶гҖӢ
  4. гҖҠiOSйҹійў‘ејҖеҸ‘(еҪ•йҹі+ж’ӯж”ҫ+еүӘиҫ‘+еҗҲжҲҗ+еҺӢзј©иҪ¬з Ғ)гҖӢ
  5. гҖҠiOSи§Ҷйў‘ејҖеҸ‘з»ҸйӘҢгҖӢ
  6. гҖҠiOS зі»з»ҹдёӯпјҢH.264 и§Ҷйў‘жөҒеҸҜд»ҘзЎ¬д»¶и§Јз Ғеҗ—пјҹ е…·дҪ“еҰӮдҪ•е®һзҺ°пјҹгҖӢ
  7. гҖҠiOSи§Ҷйў‘иҫ№дёӢиҫ№ж’ӯвҖ“зј“еӯҳж’ӯж”ҫж•°жҚ®жөҒгҖӢ
  8. гҖҠiOS з»§з»ӯж’ӯж”ҫжҲ–жҡӮж—¶е…¶д»–appзҡ„иғҢжҷҜйҹід№җгҖӢ
  9. гҖҠеҲ©з”ЁNSURLProtocolе’Ңжң¬ең°д»ЈзҗҶе®һзҺ°еңЁзәҝи§Ҷйў‘иҫ№ж’ӯж”ҫиҫ№зј“еӯҳгҖӢ
  10. гҖҠAudio Session зј–зЁӢжҢҮеҚ—(дёҖ)гҖӢ

еҸ‘иЎЁдәҺ 2016-10-03

dylib(dynamic library)пјҢWindowsзі»з»ҹзҡ„еҠЁжҖҒеә“жҳҜDLLж–Үд»¶пјҢLinuxзі»з»ҹжҳҜsoж–Үд»¶пјҢmacOSзі»з»ҹзҡ„еҠЁжҖҒеә“еҲҷдҪҝз”Ёdylibж–Үд»¶дҪңдёәеҠЁжҖҒеә“гҖӮdylibжң¬иҙЁдёҠжҳҜдёҖдёӘMach-Oж јејҸзҡ„ж–Үд»¶пјҢе®ғдёҺжҷ®йҖҡзҡ„Mach-Oжү§иЎҢж–Үд»¶еҮ д№ҺдҪҝз”ЁдёҖж ·зҡ„з»“жһ„пјҢеҸӘжҳҜеңЁж–Үд»¶зұ»еһӢдёҠдёҖдёӘжҳҜMH_DYLIBпјҢдёҖдёӘжҳҜMH_EXECUTEгҖӮ

е»әи®®ж–Үз« пјҡ
  1. гҖҠdylibжө…жһҗгҖӢ
  2. гҖҠiOS зЁӢеәҸ main еҮҪж•°д№ӢеүҚеҸ‘з”ҹдәҶд»Җд№ҲгҖӢ
  3. гҖҠdylibжіЁе…Ҙ-йҳІеҫЎгҖӢ
  4. гҖҠdylibеҠЁжҖҒеә“еҠ иҪҪиҝҮзЁӢеҲҶжһҗгҖӢ
